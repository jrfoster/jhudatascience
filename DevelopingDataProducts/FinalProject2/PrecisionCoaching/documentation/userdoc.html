Introduction

This application simulates the concept of a precision coaching application. It does this by allowing the user to visualize different aspects of the Qualitative Activity Recognition dataset generated from the study published here. It allows the user to select a test subject (athlete/patient) and then select which aspects of the data to view in a 3-D scatterplot.

The data in the dataset are complex, and so this application deals with only the subset of columns that relate to the three-dimensional measurements provided by the gyroscope, magnetometer and the accelerometer, that, in the study, were placed on several different body positions.

Details

Plot Feature Selection
In the panel on the left, you can choose which aspects of the dataset to render in the scatterplot:

  • Which test subject (athlete/patient) was performing the dumbbell curls
  • Which sensor data you want to view
  • Which body position the sensor data came from
  • Which classification of dumbbell curl the subject was performing

Plot Rendering
As you make selections in the left panel, the scatterplot is rendered on the right.

Please be aware that performance of this on Firefox and IE is less than stellar, so its recommended to use Chrome or Safari.

Also, be aware that if you clear all the activity classification checkboxes, the application will automatically select ‘Correct’ for you and will render the plot with that classification.

Notes of Interest
I think the most interesting views are the ones from the magnetometer on the arm or the forearm, but try some different combinations!
